Superstatic

Superstatic is an enhanced static web server.

Not required Apache virtual host configution :)

Options

Superstatic reads special configuration from a JSON file (either superstatic.json or divshot.json by default, configurable with -c).

{
  "name": "app-name",
  "root": "./app",
  "routes": {
    "/": "start.html"
  },
  "clean_urls": false,
  "error_page": "error.html",
  "cache_control": {
    "/*": false,
    "**/*.html": 600
  }
}

More info. See Superstatic Configuration.
